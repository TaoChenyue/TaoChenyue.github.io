## 介绍
文献管理工具。[下载地址](https://www.zotero.org/)。

## 插件
+ [ZotFile](http://zotfile.com/)： 管理PDF文件
+ [Zotero PDF Translate](https://github.com/windingwind/zotero-pdf-translate): PDF文献快捷翻译
+ [Sci-hub Plugin](https://github.com/ethanwillis/zotero-scihub)：从Sci-hub获取PDF文件
+ [DOI manager](https://github.com/bwiernik/zotero-shortdoi)：文献DOI统一
+ [Bibtex](https://retorque.re/zotero-better-bibtex/)： 文献导出为PDF文件

## 个人偏好
+ 编辑>首选项>常用，取消勾选所有文件处理
+ 编辑>首选项>同步，取消勾选所有文件同步
+ 编辑>首选项>高级>文件和文件夹，设置链接附件根目录
+ 安装浏览器插件及常用插件
